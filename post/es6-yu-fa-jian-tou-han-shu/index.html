ES6语法箭头函数

2020-09-08 1 min read

1、之前的函数定义方式

1、Function
let a = function(){  
}

2、对象字面量添加函数

let b = {
  b: function(){},
  b2(){}
}

2、箭头函数

1、放入一个参数

let a = num1 => {
    return ++num1
}

2、放入两个参数

let a = （num1, num2） => {
    return num2+num1
}

3、函数中只有一行代码

当函数中只有一行代码的时候，无论是return还是普通代码，均可以使用增强写法。

// 原来
let a = (c) => {
  return ++c
}

// 增强写法
let a = c => ++c

2、this

箭头函数的作用域是从当前位置向外一层一层找，直到找到一个this

const aa = {
  bb(){
    setTimeout(() => {
      console.log(this)
    })
  },
  cc(){
    setTimeout(function(){
      console.log(this)
    })
  }
}

aa.bb()
